The FoodGenie Android App

FoodGenie is a food delivery mobile application that allows users to browse meals, snacks, and drinks, place an order, make secure payments, track delivery, and earn rewards. The app is designed with a simple and interactive UI to enhance user experience. The FoodGenie app was created to make ordering and receiving meals easier for customers by offering a simple user-friendly smartphone interface. FoodGenie is a straightforward Android app for ordering food that lets users register, log in, and browse the menu.  This app shows how to move between several tasks in a typical food ordering procedure.

                                                         Features on the app 	
1.	User Authentication:
•	Create an account securely. The sign-up page enables users to log in or register using a Single Sign-On (SSO) button 
•	Login with your credentials to access your data.

2.	Dashboard:
Once logged in, the dashboard provides access to:
•	Menu– The user can browse through the menu before placing her order 
•	Payment Method: Users have an option to choose payment 
•	Payment conformation: After payment the user will get a message that his/her payment went through
•	Rewards – Earn and view rewards for good financial habits.
•	Track order: Get real time tracking order 
•	Profile: Can update the information in the profile  


Built With
•	Kotlin – Android App Development
•	Room Database – Local data storage
•	ViewModel + LiveData – For smooth UI and data updates
•	RecyclerView – Displaying lists of items
•	Material Design Components – Modern and intuitive UI
