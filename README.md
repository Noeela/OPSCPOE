The FoodGenie Android App

 FoodGenie is a straightforward Android app for ordering food that lets users register, log in, and browse the menu.  This app shows how to move between several tasks in a typical food ordering procedure.

  Features
 The sign-up page enables users to log in or register using a Single Sign-On (SSO) button.
 - Login Page: Enables current users to sign in.
 The menu page provides access to the Drinks, Rewards, Order Summary, and Settings pages as well as food menu selections.
 Drinks, Rewards, Order Summary, Payment, Delivery Address, and Tracking pages are among the other pages.
